Python Mini Projects Collection
This repository contains a collection of beginner-friendly Python projects developed to strengthen core programming concepts and practical problem-solving skills. The projects are console-based and focus on real-world use cases using Python fundamentals.
Projects Included
Supermarket Billing System – Menu-driven application for item selection, quantity input, bill calculation, tax computation, and invoice generation.
ATM Functionality System – Simulates basic ATM operations such as credit, debit, and balance inquiry with input validation.
Dictionary Management System – Allows users to add, search, update, delete, and display word meanings using Python dictionaries.
To-Do List Application – Task management system built using Python OOP concepts with features to add, remove, complete, and display tasks.
Key Concepts Used
Python fundamentals
Functions and Object-Oriented Programming (OOP)
Lists and Dictionaries
Loops, conditional statements, and exception handling
Menu-driven console applications
This repository reflects my hands-on learning journey in Python and showcases practical implementations of core concepts through small real-world applications.
